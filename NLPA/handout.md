## Modular Content Structure

1. **Overview of the course**
2. **Grammar and spellcheckers**
3. **Knowledge Graph Applications**
    - How to Use Knowledge Graphs
    - Why we need Knowledge Graphs
    - Traditional RAG, Graph RAG
    - Agentic RAG
4. **Question Answering and Conversational AI**
    - Question Answering
    - Conversational AI
    - Hybrid systems
    - Introduction to Gen AI techniques
    - Implementation of these systems using state of art Gen AI techniques including LLM, RAG, Agentic AI techniques
5. **Machine Translation**
    - Statistical Techniques
    - Neural Techniques
    - Indic Language Translation
6. **Information Extraction**
    - Named Entity Recognition
    - Relation Extraction
    - Extracting Events and Time
7. **Sentiment Analysis**
    - Sentiment Analysis Methods
    - Neural Networks for Sentiment Analysis
