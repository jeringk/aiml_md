# Pre-requisites — Social Media Analytics (SMA)

Topics assumed to be learnt before starting this course.

---

## Mathematics

- Probability & Statistics (distributions, hypothesis testing, descriptive stats)
- Linear Algebra basics (vectors, matrices)
- Graph theory fundamentals (nodes, edges, adjacency matrix, degree)

## Machine Learning

- Supervised Learning (classification, regression)
- Unsupervised Learning (clustering — K-Means, hierarchical)
- Feature extraction and selection
- Model evaluation metrics

## Deep Learning

- Neural Network fundamentals
- Text classification using deep learning
- Word embeddings (Word2Vec, GloVe)

## Natural Language Processing

- Text preprocessing (tokenization, stopword removal, stemming)
- Sentiment analysis basics
- TF-IDF and Bag-of-Words representations
- Topic modeling concepts (LDA basics)

## Network Science / Graph Basics

- Graph representations and properties
- Centrality measures (degree, betweenness, closeness)
- Community detection concepts

## Programming

- Python proficiency
- Data manipulation (Pandas, NumPy)
- Data visualization (Matplotlib, Seaborn)
- Familiarity with NetworkX or similar graph libraries
