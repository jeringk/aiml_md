## Repository Structure

Each course folder contains:

- **`study/`** — Topic-wise study notes and summaries
- **`questions/`** — Past question papers and generated practice questions

### Question Format

Every question follows a **3-page structure** (for print):

1. **Page 1 — Question:** The question text with marks
2. **Page 2 — Topics to Know:** Key topics to study
3. **Page 3 — Solution:** Step-by-step answer

See [`QUESTION_TEMPLATE.md`](QUESTION_TEMPLATE.md) for the exact formatting template.

---

## Formatting Conventions

- **Inline math:** `$...$` — e.g., $E = mc^2$
- **Display math:** `$$...$$`
- **Page breaks for print:**
  ```html
  <div style="page-break-after: always;"></div>
  ```
- All questions are **numbered sequentially** (Q1, Q2, Q3…)
- Formulas must be **LaTeX-rendered** and print-ready

---

## How to Use

1. Navigate to the course folder (ADL, NLPA, CAI, SMA)
2. Use `study/` for topic-wise revision
3. Use `questions/` for practice — print-friendly with page breaks
4. Open in any markdown viewer/editor that supports LaTeX math (VS Code, Typora, Obsidian, etc.)
